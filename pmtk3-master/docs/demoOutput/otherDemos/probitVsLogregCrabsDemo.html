Logistic regression

modelLR = logregFit(Xtrain, ytrain);
yhat = logregPredict(modelLR, Xtest);
nerrsLR = sum(yhat ~= ytest)

     3

Probit Regression

modelProbit = probitRegFit(Xtrain, ytrain);
yhat = probitRegPredict(modelProbit, Xtest);
nerrsProbit = sum(yhat ~= ytest)

nerrsProbit =
     3
